YO Log PRO v17.1 — CAT Edition
Logger profesional pentru concursuri de radioamatorism, scris în Python.

Funcționalități Noi (v17.1)
📡 Band Map Real: Integrare cu DX Cluster (Telnet), vizualizare spoturi în timp real.
📻 Control CAT: Suport pentru Yaesu, Kenwood, Icom, Hamlib. Citire frecvență și mod automat.
📊 Scor Live & Statistici: Calcul automat scor și rată QSO/oră.
🌍 Bază de date DXCC: Identificare automată a țărilor.
Interfață Modernă: Temă Dark/Light, design compact.
Instalare
Clonează repository-ul:
git clone https://github.com/utilizator/YOLogPRO_v17.1.git
Instalează dependențele:
bash

pip install -r requirements.txt
Rulează aplicația:
bash

python yo_log_pro.py
Utilizare
Apasă Enter pentru a loga QSO-ul.
Dublu-click pe un spot din Band Map pentru a completa automat frecvența și indicativul.
Accesează Tools -> CAT Settings pentru a configura radio-ul.
